# 📱💻 Programação Web e Mobile

Repositório criado para armazenar exercícios, projetos e práticas desenvolvidas nas disciplinas de **Programação Web** e **Desenvolvimento Mobile** durante o curso de **Desenvolvimento de Sistemas**.

## 📚 Objetivo

Organizar e documentar os códigos produzidos nas aulas, permitindo acompanhar a evolução no aprendizado de desenvolvimento para **web e dispositivos móveis**.

## 📂 Estrutura do Repositório

```
programacao-web-mobile
│
├── web
│   ├── exercicios
│   └── projetos
│
├── mobile
│   ├── exercicios
│   └── projetos
│
└── README.md
```

## 🛠 Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* PHP
* SQL
* Conceitos de desenvolvimento mobile

## 📖 Conteúdos Praticados

* Estruturação de páginas web
* Formulários e validação de dados
* Lógica de programação
* Integração com banco de dados
* Desenvolvimento de aplicações mobile
